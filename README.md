# AVENOR | Premium Web Development Portfolio

**Avenor** is a high-end web development agency specializing in conversion-focused websites for local businesses and online brands. This repository contains the source code for the official Avenor portfolio, featuring a premium dark-themed aesthetic with violet neon accents and advanced CSS animations.

## 🚀 Features
- **Ultra-Premium Design**: Dark mode aesthetic with glassmorphism and multi-layered neon glow.
- **Conversion-Optimized**: Strategically placed CTAs and high-impact hero sections.
- **Dynamic Animations**: Custom particle background, scroll-reveals, and interactive stat counters.
- **Mobile-First**: Fully responsive across all devices with a smooth UX.
- **Trust Elements**: Integrated testimonials and results-driven section hierarchy.

## 🛠️ Built With
- **Core**: HTML5, CSS3, JavaScript (Vanilla)
- **Typography**: Inter, Bebas Neue (Google Fonts)
- **Animations**: Custom CSS Keyframes & Intersection Observer API
